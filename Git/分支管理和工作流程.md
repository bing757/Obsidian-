Git Flow模型：
- main,dev是主要分支，长期存在；
- hotfix热修复分支，release发布分支，feature功能分支是辅助分支，辅助主要分支更改后可以删除，短期存在

- 功能分支feature稳定后会以某种方式合并到开发分支dev中
